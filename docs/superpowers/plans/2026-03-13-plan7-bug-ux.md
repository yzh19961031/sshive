## Chunk 1: 后端 — 会话数据一致性修复

### Task 1: cast_file_path 写入时机修复 + 僵尸会话清理

**Files:**
- Modify: `internal/audit/repo.go`
- Modify: `internal/audit/service.go`
- Modify: `cmd/server/main.go`

**背景：**
- `StartSession` 计算出 `castPath` 后只赋值给内存中的 `sess.CastFilePath`，没有写入 DB
- `CloseSession` 是通过 `defer` 调用的，服务重启时 defer 未执行，导致 `cast_file_path` 永远为空，`status` 永远为 `active`
- 修复方式：`StartSession` 创建 session 后立即用一条 UPDATE 把 `cast_file_path` 写入 DB
- 同时在启动时调用 `RecoverStaleSessions`，将所有 `status='active' AND ended_at IS NULL` 的会话更新为 `status='interrupted'`

- [ ] **Step 1: 在 repo.go 新增 UpdateCastPath 和 RecoverStaleSessions**

```go
// internal/audit/repo.go 新增以下两个方法（追加到文件末尾）

func (r *Repo) UpdateCastPath(id int64, castPath string) error {
	return db.DB.Model(&model.Session{}).Where("id = ?", id).
		Update("cast_file_path", castPath).Error
}

func (r *Repo) RecoverStaleSessions() error {
	now := time.Now()
	return db.DB.Model(&model.Session{}).
		Where("status = ? AND ended_at IS NULL", "active").
		Updates(map[string]any{
			"status":   "interrupted",
			"ended_at": now,
		}).Error
}
```

- [ ] **Step 2: 修改 service.go 的 StartSession，新增 RecoverStaleSessions**

// 复制
function copySelection() {
  const activeT = tabs.value.find(t => t.id === activeTab.value)
  if (!activeT?.term) return
  const sel = activeT.term.getSelection()
  if (sel) navigator.clipboard.writeText(sel)
}

// 主机选择
const showHostPicker = ref(false)
const hostPickerLoading = ref(false)
const pickerHosts = ref<any[]>([])

watch(showHostPicker, async (v) => {
  if (!v) return
  hostPickerLoading.value = true
  try {
    const res = await hostApi.list({ page: 1, page_size: 100 })
    pickerHosts.value = res.data.data?.list ?? []
  } finally {
    hostPickerLoading.value = false
  }
})

function pickHost(h: any) {

// 状态 tag 颜色辅助
function statusTag(status: string) {
  const map: Record<string, 'success' | 'warning' | 'default' | 'error'> = {
    active: 'success',
    interrupted: 'warning',
    closed: 'default',
  }
  return map[status] ?? 'default'
}

// 时长格式化
function formatDuration(row: any) {
  const end = row.ended_at ? new Date(row.ended_at) : new Date()
  const diff = Math.floor((end.getTime() - new Date(row.started_at).getTime()) / 1000)
  if (diff < 60) return `${diff}s`
  if (diff < 3600) return `${Math.floor(diff / 60)}m ${diff % 60}s`
  return `${Math.floor(diff / 3600)}h ${Math.floor((diff % 3600) / 60)}m`
}

const columns = [
