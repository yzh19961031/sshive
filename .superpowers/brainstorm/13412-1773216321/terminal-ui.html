分屏模式

支持将多个 Terminal 分屏显示，同时观察多台服务器状态。适合需要对比监控的场景。

prod-web-01
prod-db-01
⊞ 分屏
root@web:~# nginx -t
syntax is ok
root@web:~# █
root@web:~# tail -f app.log
[INFO] Server start
[ERROR] DB timeout

Tab + 分屏都支持

默认 Tab 模式，工具栏提供分屏按钮，可随时切换。兼顾日常使用和高级场景，灵活性最强。
